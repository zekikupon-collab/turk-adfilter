## uBlock Origin'de Kullanım

1. **Ayarları Açın**: uBlock Origin ayarlarını açın
2. **Filtreler Sekmesi**: "Filtreler" sekmesine tıklayın
3. **İçe Aktarın**: "Özel" bölümünde "İçe Aktar" butonuna tıklayın
4. **Bağlantıyı Ekleyin**: Yukarıdaki RAW bağlantılarından birini yapıştırın ve "Değişiklikleri Uygula" butonuna tıklayın

![uBlock Origin Kurulum](/assets/ublock-kurulum.png)

## AdGuard'da Kullanım

1. **Ayarları Açın**: AdGuard ayarlarını açın
2. **Filtreler Bölümü**: "Filtreler" > "Özel Filtreler" sekmesine gidin
3. **Filtre Ekle**: "Özel Filtre Ekle" butonuna tıklayın
4. **Bağlantıyı Ekleyin**: Yukarıdaki RAW bağlantılarından birini yapıştırın ve "Ekle" butonuna tıklayın

## Pi-hole'da Kullanım

1. **Yönetim Paneline Giriş**: Pi-hole yönetim panelinize giriş yapın
2. **Adlist Ekleme**: "Group Management" > "Adlists" sekmesine gidin
3. **Liste Ekle**: "Add a new adlist" kısmına aşağıdaki Pi-hole bağlantılarından birini yapıştırın
4. **Ekle ve Güncelle**: "Add" butonuna tıklayın ve ardından "Update Gravity" ile güncellemeleri uygulayın

## DNS Çözümleyici Formatları

Her release'de dnsmasq, Unbound ve BIND RPZ formatlarında dosyalar yayınlanır. [Son release](https://github.com/omerdduran/turk-adfilter/releases/latest) sayfasından indirebilirsiniz.

### dnsmasq

1. **Dosyayı İndirin**:
```bash
curl -o /etc/dnsmasq.d/turk-adfilter.conf https://github.com/omerdduran/turk-adfilter/releases/latest/download/dnsmasq.conf
```
2. **Servisi Yeniden Başlatın**:
```bash
sudo systemctl restart dnsmasq
```

### Unbound

1. **Dosyayı İndirin**:
```bash
curl -o /etc/unbound/turk-adfilter.conf https://github.com/omerdduran/turk-adfilter/releases/latest/download/unbound.conf
```
2. **Yapılandırmaya Ekleyin**: `unbound.conf` dosyanıza şu satırı ekleyin:
```
include: /etc/unbound/turk-adfilter.conf
```
3. **Servisi Yeniden Başlatın**:
```bash
sudo systemctl restart unbound
```

### BIND (RPZ)

1. **Zone Dosyasını İndirin**:
```bash
curl -o /etc/bind/zones/turk-adfilter.rpz https://github.com/omerdduran/turk-adfilter/releases/latest/download/rpz.zone
```
2. **Yapılandırmaya Ekleyin**: `named.conf` dosyanıza şu blokları ekleyin:
```
response-policy { zone "turk-adfilter.rpz"; };

zone "turk-adfilter.rpz" {
    type master;
    file "/etc/bind/zones/turk-adfilter.rpz";
};
```
3. **Servisi Yeniden Başlatın**:
```bash
sudo systemctl restart named
```

## Mobil Cihazlarda Kullanım

### Android

> Android cihazlarda reklamları engellemek için aşağıdaki uygulamaları kullanabilirsiniz:

- **AdGuard for Android**: Uygulamayı yükleyin ve yukarıdaki AdGuard talimatlarını takip edin
- **Blokada**: Uygulamayı yükleyin ve özel liste olarak RAW bağlantılarından birini ekleyin
- **DNS66**: Uygulamayı yükleyin ve özel host kaynağı olarak RAW bağlantılarından birini ekleyin

### iOS

> iOS cihazlarda reklamları engellemek için aşağıdaki yöntemleri kullanabilirsiniz:

- **AdGuard for iOS**: Uygulamayı yükleyin ve ayarlardan özel liste olarak RAW bağlantılarından birini ekleyin
